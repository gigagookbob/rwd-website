Your AI coding sessions, journaled by day.

rwd is an open-source CLI for Claude Code and Codex that rewinds each session into a daily journal with work summaries, key decisions, model corrections, and shareable updates.

Session Capture

Scan the day in one command

Run rwd today to discover sessions across Claude Code and Codex, then turn the day into one structured journal entry.

Terminal
$ rwd today
⠋ Discovering sessions...
[discover] 14 projects, 82 log files scanned
Codex sessions: 8
⠋ Analyzing insights...
▸ Session: 019d86xx
Summary: release workflow was finalized...
Markdown saved: ~/Daily/2026-04-13.md
Structured Journal

Same outline, every single day

rwd keeps each day readable with stable sections: Work Summary, Key Decisions, Questions, and Model Corrections.

2026-04-13.md
## Work Summary
- Stabilized release flow and validated checks.
## Key Decisions
- Root-cause fix over workaround
## Questions & Curiosities
- Should automated sessions be included by default?
## Model Corrections
- Model: Context7 was unavailable
Fix: Retried after MCP connection
Corrections & Learning

Capture what changed your mind

rwd highlights correction loops and lessons learned, so your journal records why the final solution changed.

Model vs Fix
Model: "Local bubblewrap binary should be enough."
Fix: Set setuid on /usr/bin/bwrap.
---
TIL: PATH duplication can mimic duplicate installs.
TIL: Existing tags can skip automated release workflows.
Privacy Guardrails

Redaction runs before analysis

Sensitive values are masked locally first, so your daily journal keeps context while reducing accidental exposure.

Redactor
Input
OPENAI_KEY=sk-...
Bearer eyJhbGciOi...
Server: 192.168.1.42
# Redacted Output
## Before LLM Analysis
OPENAI_KEY=[REDACTED:API_KEY]
[REDACTED:BEARER_TOKEN]
Server: [REDACTED:PRIVATE_IP]
## Summary
- Redacted: API_KEY, BEARER_TOKEN, PRIVATE_IP
Daily Summary

Generate a shareable day recap

rwd summary condenses sessions into a clean progress report grouped by topic. Good for personal logs and standup prep.

Terminal
$ rwd summary --lang en
 
=== Development Progress ===
### Codex Environment
- Removed recurring bubblewrap warning.
 
### rwd
- Completed release flow for v0.13.10.
 
✓ Copied to clipboard
Slack Handoff

Post updates without rewriting

rwd slack turns your journal into a team-ready update and copies it directly to clipboard.

Terminal
$ rwd slack --lang en
 
[Today's Work Update]
- Session classification flow stabilized.
- Release workflow validated and completed.
- Security guardrails and checks were improved.
 
✓ Copied to clipboard

Install in seconds

Linux / macOS / WSL2 (Linux shell) curl -fsSL https://raw.githubusercontent.com/gigagookbob/rwd/main/install.sh | sh

Native Windows is not supported. Please use WSL2.
